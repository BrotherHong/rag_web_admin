# 檔案類型圖示功能

## 新增功能

為上傳檔案頁面的所有檔案列表添加了根據檔案類型顯示不同顏色圖示的功能。

## 支援的檔案類型與圖示

### 📕 PDF 檔案 (.pdf)
- **顏色**: 紅色 (text-red-500)
- **用途**: PDF 文件

### 📘 Word 文件 (.doc, .docx)
- **顏色**: 藍色 (text-blue-500)
- **用途**: Microsoft Word 文件

### 📗 Excel 試算表 (.xls, .xlsx)
- **顏色**: 綠色 (text-green-500)
- **用途**: Microsoft Excel 試算表

### 📙 PowerPoint 簡報 (.ppt, .pptx)
- **顏色**: 橘色 (text-orange-500)
- **用途**: Microsoft PowerPoint 簡報

### 📄 文字檔案 (.txt)
- **顏色**: 灰色 (text-gray-500)
- **用途**: 純文字檔案

### 📋 其他檔案類型
- **顏色**: 淺灰色 (text-gray-400)
- **用途**: 所有其他未特別定義的檔案類型

## 實作細節

### 1. 核心函數：`getFileIcon(fileName)`

```javascript
const getFileIcon = (fileName) => {
  const ext = fileName.split('.').pop().toLowerCase();
  
  // 根據副檔名返回對應的 SVG 圖示
  // ...
}
```

**功能**:
- 自動解析檔案名稱取得副檔名
- 根據副檔名返回相應顏色和類型的 SVG 圖示
- 不區分大小寫（自動轉為小寫）

### 2. 應用位置

#### 步驟 1: 已選擇的檔案列表
- **尺寸**: w-10 h-10 (40x40 像素)
- **位置**: 檔案名稱左側
- **用途**: 直觀顯示檔案類型

```jsx
<div className="flex-shrink-0 mr-4">
  {getFileIcon(file.name)}
</div>
```

#### 步驟 2: 重複檢查結果
- **尺寸**: 縮放 75% (scale-75)
- **位置**: 檔案名稱左側
- **用途**: 在重複檢查時識別檔案類型

```jsx
<div className="flex-shrink-0 mr-3">
  <div className="scale-75">
    {getFileIcon(result.fileName)}
  </div>
</div>
```

#### 步驟 4: 結果摘要 - 失敗檔案列表
- **尺寸**: 縮放 50% (scale-50)
- **位置**: 錯誤圖示和檔案名稱之間
- **用途**: 在失敗列表中輔助識別

```jsx
<svg className="w-5 h-5 text-red-500 mr-2" />  {/* 失敗圖示 */}
<div className="scale-50 -ml-2">
  {getFileIcon(file.name)}
</div>
<p>{file.name}</p>
```

#### 步驟 4: 結果摘要 - 成功檔案列表
- **尺寸**: 縮放 50% (scale-50)
- **位置**: 成功圖示和檔案名稱之間
- **用途**: 在成功列表中輔助識別

```jsx
<svg className="w-5 h-5 text-green-500 mr-2" />  {/* 成功圖示 */}
<div className="scale-50 -ml-2">
  {getFileIcon(file.name)}
</div>
<p>{file.name}</p>
```

## 優勢

1. **視覺識別**: 使用者可以快速識別檔案類型
2. **色彩編碼**: 不同類型使用不同顏色，符合直覺
3. **一致性**: 所有列表使用相同的圖示系統
4. **響應式**: 在不同步驟中適當調整大小

## 擴展性

如果需要添加新的檔案類型支援，只需在 `getFileIcon` 函數中添加新的判斷：

```javascript
// 範例：添加 CSV 支援
if (ext === 'csv') {
  return (
    <svg className="w-10 h-10 text-teal-500" fill="currentColor" viewBox="0 0 20 20">
      <path fillRule="evenodd" d="..." clipRule="evenodd" />
    </svg>
  );
}
```

## 瀏覽器兼容性

- ✅ SVG 圖示：所有現代瀏覽器完全支援
- ✅ Tailwind CSS 類別：使用標準工具類別
- ✅ CSS Transform (scale)：廣泛支援

## 性能考量

- **輕量級**: 使用純 SVG，不需要外部圖片
- **快速渲染**: 內聯 SVG 不需要額外的網路請求
- **記憶體友善**: 函數式組件，按需渲染
